Copy cookies from browser DevTools

  1. Open royalbadminton.ezfacility.com and log in.
  2. Press F12 (or Cmd+Option+I on Mac) to open DevTools.
  3. Go to Network tab → reload the page → click any request to royalbadminton.ezfacility.com.
  4. In the Headers panel, find Request Headers → Cookie.
  5. Copy the entire Cookie header value and paste it below.

The cookie header contains all cookies at once, e.g. ASP.NET_SessionId=abc; .ASPXAUTH=xyz; ...
